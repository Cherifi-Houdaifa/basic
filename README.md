# basic
A simple basic like compiler to c.

## Language syntax
```
PRINT "Hello World" ; 
LET max ;
LET i ;
INPUT max ;
IF max > 20 THEN
    max = 20 ;
ENDIF
WHILE i < max DO
    PRINT i ;
ENDWHILE
```

## Language features
- printing strings
- defining variabels (only numbers)
- taking inputs (only numbers)
- if statements
- while loops
